Un archivo .map es un fichero intermedio que establece un mapping entre un lenguaje de progamación y otro. Lo encontramos, por ejemplo, con SASS, dado que no se compila directamente de SASS a CSS, sino que primero se utiliza una compilación o traducción intermediaria utilizando un fichero .map

Pues lo mismo pasa con TS y JS cuando vamos a publicar nuestra app de Angular. Esta se pasa de TS a JS, pero no de forma inmediata, sino usando ficheros .map también
