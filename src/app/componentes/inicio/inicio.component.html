Propósito

  • Esta web tiene como objetivo ilustrar el funcionamiento de Angular de forma sencilla, breve y directa
  • Me he formado en el funcionamiento de este framework durante un largo periodo de tiempo y he creado estas notas por si le pudieran servir a alguien basándome en diferente fuentes

Introducción

  • Angular es un framework de front desarrollado por Google que utiliza TypeScript
  • Un framework es un conjunto de herramientas software que quedan a disposición del programador para elaborar aplicaciones web contando con un gran abanico de prestaciones que no tendrías a tu disposición programando desde cero
  • TypeScript es un lenguaje de programación desarrollado por Microsoft que utiliza JavaScript como base crear un lenguaje de programación más robusto
  • Ofrece numerosas prestaciones adicionales con las que JavaScript no contaba en principio, por lo que se conoce como un superconjunto de JavaScript
  • Algunas de dichas prestaciones vienen a ser el tipado de variables, notificación de errores previa al periodo de ejecución, etc.
