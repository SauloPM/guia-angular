1. Introducción

1.1 Qué es Auth0

  • Auth0 es una empresa encargada del desarrollo de logins robustos para aplicaciones web y móviles a través de múltiples herramientas
  • Esta plataforma ofrece numerosas herramientas, como la autentificación de doble factor, login utilizando nuestras cuentas en redes sociales, recuperación de contraseña, envío de contraseña de validación, etc.
  • Muchas compañías famosas a nivel internacional hacen uso de las herramientas proporcionadas por esta empresa, como NVIDIA o Mozilla Firefox, entre otras
  • Es un servicio de pago, pero ofrece un abanico de opciones más que aceptables si deseas utilizarlo gratuitamente

1.2 Registro

  • Para poder utilizar este servicio, debemos registrarnos en su página web
  • Dentro del formulario de registro, debemos escribir un ID único y seleccionar la región más cercana a nosotros de entre las disponibles
  • Completar el resto del formulario es bastante sencillo

1.2 Crear proyecto

  • Una vez registrados, dentro del dashboard, hacemos clic en Create Application
  • Tras esto, escribimos el nombre de nuestra aplicación y seleccionamos Single Page Web Applications
  • Nótese que las otras opciones corresponden a aplicaciones nativas o aplicaciones web regulares, entre otras
  • En cada opción aparecen las herramientas comúnmente asociadas a cada caso
  • Por ejemplo, en la que hemos seleccionado aparecen Angular, AngularJS o NodeJS, lo que nos puede dar una pista de qué opción seleccionar en base a las herramientas que estemos utilizando
  • Posteriormente, seleccionamos Angular y aparecerá un quickstart de gran utilidad

1.4 Integrar Auth0 en nuestra aplicación web

  • Para integrar Auth0 en nuestra aplicación, necesitamos la librería auth0-spa-js, que podemos integrar en nuestro proyecto de dos formas: utilizando el gestor de paquetes npm o el CDN
  • Lógicamente, de escoger la primera opción, se modificarían los ficheros package.json y package-lock.json y tendríamos que importar esta librería (instrucción import)
  • A continuación, creamos un servicio en nuestro proyecto ejecutando el comando $ ng g s servicios/auth
  • Dentro del fichero TS, pegamos el código que aparece en el quickstart
